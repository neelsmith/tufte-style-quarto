# A Tufte-like design for Quarto books

This repository defines a [Quarto](https://quarto.org) book project configured to produce both a web site and a PDF version of the text with `quarto render`.

See the current project documentation as [a PDF](https://github.com/neelsmith/tufte-style-quarto/blob/main/A-Quarto-book-project-with-Tufte-style-layout.pdf?raw=true), or on [github pages](https://neelsmith.github.io/tufte-style-quarto/).
